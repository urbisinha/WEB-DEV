This project is a one-page responsive portfolio website designed and developed as part of Lab 2. The website incorporates Flexbox, CSS Grid, media queries, and CSS animations to create a modern, polished, and interactive layout. It follows a mobile-first approach and adapts smoothly across mobile, tablet, and desktop devices.
The website includes the following sections:
Header, Hero, About, Skills, Projects, Contact, and Footer, each styled using modern CSS concepts like transitions, hover effects, and keyframe animations.
